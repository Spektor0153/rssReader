# rssReader

Читалка новостных RSS лент. Парсинг XML файлов при запросе новостной ленты, список адресов каналов хранится в Mongo DB.

Use: 
<ul>
	<li>Node js</li>
	<li>Mongo DB</li>
	<li>Express</li>
	<li>Socket.io</li>
	<li>XML parser</li>
	<li>Webpack</li>
	<li>React</li>
	<li>Redux</li>
	<li>Bootstrap</li>
	<li>Centos 6</li>
</ul>
